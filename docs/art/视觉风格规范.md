# 视觉风格规范 v0（五藏山经）

## 方向

- **主体风格**：像素风（角色/可交互物）。
- **背景风格**：旷野水墨（横版、超宽、留白与层雾）。
- **镜头感**：横版自由探索，背景宽于常规关卡屏宽，强调“山海旷野”。

## 分层规则

1. **前景交互层**：像素风地物、角色、UI 提示。
2. **中景叙事层**：稀疏山石、枯木、遗迹轮廓。
3. **远景气氛层**：水墨山脊、雾带、低对比大形体。

## 尺寸基线

- 默认异兽：**32×32 像素**。
- 大型异兽：在条目中单独标注（例：48×48、64×64），必要时保留 16×16 缩略草图用于行为验证。
- 背景：推荐 16:9 基础上做 **2x~4x 横向延展**（如 2560×720 / 3840×720）。
