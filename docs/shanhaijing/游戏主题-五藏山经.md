# 游戏主题范围：五藏山经

本项目的**叙事与关卡主轴**定为《山海经》中的 **《山经》五篇**，即传统所称 **「五藏山经」**（或「五经」）：

| 顺序 | 篇名（通行称） | 说明 |
|------|----------------|------|
| 1 | **《南山经》** | 南方山系与泽薮；湿热林泽、犀象与怪鸟异鱼较多。 |
| 2 | **《西山经》** | 西方山系；多金玉、大泽与西王母神话群。 |
| 3 | **《北山经》** | 北方山系；寒温河谷、大河支流与「北冥」意象延伸。 |
| 4 | **《东山经》** | 东方山系；滨海、沙洲、岛链与潮汐带。 |
| 5 | **《中山经》** | 中央山系；伊洛一带山川祠祀密集，博物与「天下之中」感强。 |

## 与全书其余部分的关系

《山海经》全书还包括 **《海经》《大荒经》** 等篇。仓库数据表中可能仍保留少量 **海经 / 大荒** 条目作美术或系统参考，但在**主题定位**上它们属于 **扩展资料**，不纳入上述五篇的主线经本；后续剧情、地图分区、肉鸽「书卷」解锁等，建议以 **五篇山经** 为纲做章节划分。

## 数据与机读

`tools/render_shanhaijing_docs.py` 在 `game/data/creatures_catalog.json` 中写入 **`theme_scope`** 字段，统计 `jing_zh` 属于上述五篇的条目数与全表总数之比，供 Godot 与策划核对「主线覆盖率」。

## 文本与用字

篇名在数据层统一为 **`jing_zh` 字段**（如 `南山經`、`东山經`）；「藏」「经」等异写、繁简以你们选定的整理本为准，宜在全项目统一一次。
