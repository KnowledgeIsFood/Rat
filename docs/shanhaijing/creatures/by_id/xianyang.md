- **稳定 ID**：`xianyang`
- **所属典籍**：五藏山經 · 西山經

- **所在山川 / 水域**：錢來之山

## 文述画像（据经籍大意）

其状如羊而马尾。

## 特点与异效（经载 / 传说）

其脂可以已腊（去皴裂/润肤类），可做药剂素材。

多松、洗石出焉；干燥松针酸性土与裸岩。

## 生物链与种间关系（推演）

草食：禾草、灌木；狼、豹、金雕捕食。
