# 耳鼠

- **稳定 ID**：`ershu`
- **类目**：异兽
- **所属典籍**：五藏山經 · 北山經
- **篇段 / 列系**：北次三經
- **所在山川 / 水域**：丹熏之山

## 文述画像（据经籍大意）

状如鼠，而菟首麋身，其音如嗥犬，以其尾飞。

## 特点与异效（经载 / 传说）

食之不䐆，又可以御百毒（抗毒/缓饿类肉鸽祝福）。

## 生态环境（推演）

多臭椿、多红白垩；干燥坡地与灌丛。

## 生物链与种间关系（推演）

杂食：种子、昆虫、菌类；猛禽、鼬科捕食。

- **营养级角色（设计标签）**：初级—次级消费者

## 美术画像要点（由文言转化为可视要素）

鼠体麋身，兔耳般首，尾展如小型滑翔翼。

## 关联

- 丹熏之山
