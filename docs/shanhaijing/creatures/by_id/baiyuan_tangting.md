# 白猿

- **稳定 ID**：`baiyuan_tangting`
- **所属典籍**：五藏山經 · 南山經

- **所在山川 / 水域**：堂庭之山

## 文述画像（据经籍大意）

多白猿；猿似猕猴而大臂、脚长便捷，色有黑有黄，此处特举白色型。

## 特点与异效（经载 / 传说）

经未直言药效；可作高机动、善攀援的林冠层生物。

## 生态环境（推演）

多棪木、多水玉与黄金的山地；林冠浓密、湿度较高。

## 生物链与种间关系（推演）

果实与嫩叶为主；与大型猫科、雕类存在捕食压力关系。
