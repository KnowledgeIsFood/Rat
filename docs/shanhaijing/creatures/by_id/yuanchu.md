# 鵷鶵

- **稳定 ID**：`yuanchu`
- **类目**：奇禽
- **所属典籍**：五藏山經 · 南山經
- **篇段 / 列系**：南次三經
- **所在山川 / 水域**：南禺之山（佐水）

## 文述画像（据经籍大意）

凤属，与凤凰同现于南禺之山穴水冬闭夏出之处。

## 特点与异效（经载 / 传说）

祥瑞从属色：非凤凰本体但同系，可做稀有配色异变。

## 生态环境（推演）

多金玉其上下多水；海蚀穴潮汐通道。

## 生物链与种间关系（推演）

神话从属：与凤皇共享“饮啄自然”叙事，少涉凡俗捕食。

- **营养级角色（设计标签）**：文化符号位

## 美术画像要点（由文言转化为可视要素）

较凤皇更青紫的羽色，体型略小；穴水喷涌如呼吸节律。

## 关联

- 鳳皇

## 资料说明

据《南山經》南次三經南禺之山条。
