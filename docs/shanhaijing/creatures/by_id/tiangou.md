# 天狗

- **稳定 ID**：`tiangou`
- **类目**：异兽
- **所属典籍**：五藏山經 · 西山經
- **篇段 / 列系**：西次三經
- **所在山川 / 水域**：阴山

## 文述画像（据经籍大意）

其状如狸而白首，其音如榴榴，可以御凶。

## 特点与异效（经载 / 传说）

御凶类护符生物；可做随从或结界反伤。

## 生态环境（推演）

阴山多砺石、洗石；干燥硬叶灌丛与裸岩。

## 生物链与种间关系（推演）

肉食偏小型：啮齿、蜥蜴；白首在岩区具隐蔽优势。

- **营养级角色（设计标签）**：次级消费者

## 美术画像要点（由文言转化为可视要素）

狸形白首，榴榴声化作护符光环；砺石如齿列背景。

## 关联

- 阴山洗石
