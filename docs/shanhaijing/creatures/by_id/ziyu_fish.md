- **类目**：鳞介

- **篇段 / 列系**：东次一經
- **所在山川 / 水域**：东次一經水系（妑水一带，各本略异）

## 文述画像（据经籍大意）

状如鲋而赤鳞，其音如叱。

## 特点与异效（经载 / 传说）

食之不骄（情绪类），可做镇静类药剂。

## 生态环境（推演）

多青雘、多赤鱬类近水域；暖水浅沼。

## 生物链与种间关系（推演）

杂食小鱼：藻类、无脊椎动物；鹭、蛇、大型鱼捕食。

鲫形赤鳞，叱声化为水纹冲击波；与南山赤鱬可做跨域变体。
