# 猾褢

- **稳定 ID**：`huaihuai`
- **所属典籍**：五藏山經 · 南山經
- **篇段 / 列系**：南次二經
- **所在山川 / 水域**：堯光之山

## 文述画像（据经籍大意）

状如人而彘鬣，穴居而冬蛰，其音如斲木。

## 特点与异效（经载 / 传说）

见则县有大繇（徭役大兴），可做“劳役/大工程”叙事压力。

## 生态环境（推演）

山南多玉、山北多金；洞系发育的温带山地。

## 生物链与种间关系（推演）

杂食偏掘根：块根、昆虫、小型脊椎动物；冬眠降低冬季能流。

- **营养级角色（设计标签）**：次级消费者

## 美术画像要点（由文言转化为可视要素）

人形覆以野猪鬃，爪持木屑飞溅；洞口霜线示冬蛰。

## 关联
