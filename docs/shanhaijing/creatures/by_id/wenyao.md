# 文鳐鱼

- **稳定 ID**：`wenyao`
- **类目**：鳞介
- **所属典籍**：五藏山經 · 西山經
- **篇段 / 列系**：西次三經
- **所在山川 / 水域**：泰器之山（观水）

## 文述画像（据经籍大意）

鱼身鸟翼，苍文白首赤喙，常行西海游于东海，以夜飞。

## 特点与异效（经载 / 传说）

食之已狂，见则天下大穰（癫狂治疗与丰年双标签）。

## 生态环境（推演）

观水西流注流沙；海陆风夜飞路线。

## 生物链与种间关系（推演）

浮游生物与小鱼；海鸟、大型鱼捕食成体。

- **营养级角色（设计标签）**：中级消费者 / 跨区域迁徙者

## 美术画像要点（由文言转化为可视要素）

翼展如燕鱼，夜飞拖星屑；西海蓝与东海青绿渐变。
