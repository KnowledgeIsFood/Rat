# 蠱雕

- **稳定 ID**：`gudiao`
- **所属典籍**：五藏山經 · 南山經

- **所在山川 / 水域**：鹿吳之山（泽更之水）

## 文述画像（据经籍大意）

状如雕而有角，其音如婴儿之音，是食人。

## 特点与异效（经载 / 传说）

空中顶级威胁；婴儿啼声作诱饵机制。

## 生态环境（推演）

多金石无草木的裸山与急流；强风上升气流利于猛禽巡猎。

## 生物链与种间关系（推演）

食人亦捕食中小型兽与鱼；与彘、九尾狐同属“高危异兽”生态顶层。
