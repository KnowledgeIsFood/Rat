# 玃如

- **稳定 ID**：`jueru`
- **类目**：异兽
- **所属典籍**：五藏山經 · 西山經
- **篇段 / 列系**：西次四經
- **所在山川 / 水域**：皋涂之山

## 文述画像（据经籍大意）

其状如鹿而白尾，马足人手而四角。

## 特点与异效（经载 / 传说）

经未载服食之效；适合作为高山草甸线的稀有坐骑/精英怪。

## 生态环境（推演）

皋涂多玃如、多麋；高山林线附近寒湿草甸。

## 生物链与种间关系（推演）

草食为主：莎草、灌木嫩枝；狼、豹、雕捕食。

- **营养级角色（设计标签）**：大型初级消费者

## 美术画像要点（由文言转化为可视要素）

鹿体白尾，人手马蹄的错位拼接；四角如冠。
