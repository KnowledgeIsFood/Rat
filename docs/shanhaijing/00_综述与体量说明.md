# 《山海经》综述与本书体量

## 本书性质（极简）

《山海经》非单一文体，而是 **山经（近似山川博物志）** 与 **海经 / 大荒经（神话地理与异国叙事）** 的汇编。历代对其性质判定从“地理”“数术”“巫书”“小说之祖”到现代神话学、考古与历史地理学，结论多元。**本游戏资料层取“博物 + 神话 + 可玩化”立场**：先忠实呈现通行本中的形体与效验，再允许为玩法做有标注的推演。

## 体量与当前收录范围

全书动物、神兽、异人、复合异怪数量极大（不同统计口径从百余到数百不等）。本仓库 **v1 已结构化收录 60+ 条**（见 `CATALOG.md`），覆盖：

- **五藏山经**：南、西、北、东、中山的部分代表性异兽与水族；
- **海经**：海内/海外若干高辨识条目（帝江、刑天、相柳、巴蛇、陆吾、开明兽等）；
- **大荒经**：九婴、夸父、应龙、夔等强叙事单位。

后续迭代建议：

1. 以 **山列为单位** 批量补全（每条山系一张表），减少遗漏。
2. 对 **同名异物** 与 **异文** 建 `alias` 与 `disambiguation` 字段（机读层扩展）。
3. 与 **郭璞注 / 郝懿行笺疏 / 袁珂校注** 等选定注本对齐关键句读。

## 「画像」说明

古图不存，明清绘本亦与经文时有出入。资料层中的 **画像** = **经籍形体描写（白话整理） + 美术可控要点**，用于原画与建模；不是文物鉴定结论。
