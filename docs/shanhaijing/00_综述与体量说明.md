# 《山海经》综述与本书体量

## 本书性质（极简）

《山海经》非单一文体，而是 **山经（近似山川博物志）** 与 **海经 / 大荒经（神话地理与异国叙事）** 的汇编。历代对其性质判定从“地理”“数术”“巫书”“小说之祖”到现代神话学、考古与历史地理学，结论多元。**本游戏资料层的主线**取 **《山经》五篇——五藏山经**（《南山经》《西山经》《北山经》《东山经》《中山经》）为纲；海经、大荒等可作扩展参考，见 [游戏主题-五藏山经.md](游戏主题-五藏山经.md)。

## 体量与当前收录范围

全书动物、神兽、异人、草木、复合异怪数量极大（不同统计口径从数百到上千不等）。本仓库主表 `creatures_master.csv` 经 **类目（`kind_zh`）** 区分 **奇禽 / 异兽 / 鳞介 / 草木** 等，并已用脚本批量追加占位行以抬高条目数（见 `CATALOG.md`）；**仍不等于「全书逐条校勘补全」**，后续应据选定注本逐条替换占位、核对异文。

- **五藏山经**：南、西、北、东、中山的异兽与 **草木专表**（`plants_*` JSON）；
- **海经**：海内/海外若干高辨识条目；
- **大荒经**：强叙事单位。

后续迭代建议：

1. 以 **山列为单位** 批量补全（每条山系一张表），减少遗漏。
2. 对 **同名异物** 与 **异文** 建 `alias` 与 `disambiguation` 字段（机读层扩展）。
3. 与 **郭璞注 / 郝懿行笺疏 / 袁珂校注** 等选定注本对齐关键句读。

古图不存，明清绘本亦与经文时有出入。资料层中的 **画像** = **经籍形体描写（白话整理） + 美术可控要点**，用于原画与建模；不是文物鉴定结论。
